# Alarm Roulette

This react application if for the people that find it hard to get out of bed!

## How to use:

1. Clone this repo
2. `cd AlarmRoulette`
3. Run `npm install` to install dependencies
4. After install run `npm run build`
5. Run `npm start` and head over to `localhost:8080`

## What happens if i don't get up?

* If you don't get out of bed and turn the alarm off it will send out a message to slack.
* The more embarrassing the message you enter into the alarm, the more you will want to get out of bed!
